Modern Programming Language (ISY208)

In ISY208, students focus on developing programming skills and practical applications in the field of user interface development and web programming. The main topics include:

1. Java GUI Programming

  • Learning to create Graphical User Interface (GUI) applications using Java Swing and other libraries such as AWT.
  • Designing windows, buttons, menus, tables, and interactive forms.
  • Handling events and managing user interactions with the application.

2. Introduction to Web Programming

  • Learning the basics of HTML to structure web pages.
  • Using CSS to style and design attractive web pages.
  • Adding interactivity and dynamic behavior with JavaScript, such as form validation or interactive page elements.

3. Practical Integration

  • Combining theoretical concepts with small projects, such as desktop applications with GUI or simple websites.
  • Emphasizing organization, design, and user experience, alongside writing clean and readable code.
